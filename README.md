# easy-pdk-programmer-hardware
Easy PDK programmer for PADAUK microcontroller. EDA, schematic, gerber, bom, housing stl, firmware

![Easy PDK Programmer](easypdkprogrammer.jpg?raw=true "Easy PDK Programmer")
